Management System

Веб-приложение для автоматизации управления проектами и задачами, разработанное как часть технического задания.
Основные возможности

    Управление проектами:

        Пошаговое создание проектов (5 шагов).

        Интеграция с API для выбора исполнителей и руководителей.

        CRUD операции: создание, просмотр, редактирование и удаление проектов.

    Управление задачами:

        Создание задач с привязкой к конкретному проекту.

        Добавление комментариев к задачам.

        Интерактивное управление статусами (ToDo, InProgress, Done).

        Фильтрация задач по их текущему статусу.
        

Стек технологий

    Frontend: Vue.js 3 (Composition API), Tailwind CSS.

    Backend: ASP.NET Core Web API (C#).

    Архитектура: RESTful API.

Запуск проекта
Backend

    Убедитесь, что установлена среда .NET SDK.

    Перейдите в директорию API проекта.

    Запустите сервер командой:

dotnet run

4. API должен быть доступен по адресу `http://localhost:5226`.

### Frontend
1. Откройте файл `index.html` в любом современном браузере.
2. Убедитесь, что бэкенд запущен, чтобы приложение могло получать данные через API.

## Описание интерфейса
Интерфейс разделен на логические блоки:
* **Визард (Wizard):** Пошаговая форма для создания нового проекта.
* **Панель проектов:** Таблица всех созданных проектов с возможностью их удаления.
* **Панель задач:** Инструментарий для добавления задач и мониторинга их текущего состояния.
